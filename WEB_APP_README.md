# Web应用使用说明

## 启动Web应用

运行以下命令启动Web服务器：

```bash
python3 web_app.py
```

然后在浏览器中访问：http://localhost:5000

## 功能说明

### 1. 用户输入框
- 在文本框中输入您的产品需求
- 点击"开始编排"按钮启动多智能体编排流程

### 2. 执行过程展示
- **流程图**：实时显示各个Agent的执行状态
  - 📚 产品研究员
  - 📝 文档助手
  - 🔍 可行性评估
  - 🎯 结果汇总
- **执行日志**：显示详细的执行过程日志

### 3. 最终输出结果
- **最终汇总**：显示可行性评分、价值主张、成功因素等
- **详细结果**：通过标签页查看各个Agent的详细输出
  - 产品研究结果
  - 产品文档
  - 可行性评估
  - 最终汇总

## 技术栈

- **后端**：Flask (Python)
- **前端**：HTML5, CSS3, JavaScript (原生)
- **样式**：现代化渐变设计，响应式布局

## 注意事项

- 确保所有依赖已安装：`pip3 install -r requirements.txt`
- 确保 `config.py` 中的API配置正确
- 执行过程可能需要一些时间，请耐心等待
